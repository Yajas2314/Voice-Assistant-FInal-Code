# 🤖 Zephyr 2.0 - AI Assistant

## Features
- 🔐 Secure login system (hashed password)
- 🌦 Weather API integration
- 🌐 Web search & app launcher
- 🎵 Spotify access
- 🤖 AI assistant (ChatGPT-like)
- 📜 Command logging

## Run

pip install -r requirements.txt
streamlit run app.py

## Setup

- Add your OpenAI API key:
  export OPENAI_API_KEY=your_key

- Add Weather API key in code

## Commands

- time
- weather
- search <query>
- open chrome
- open notepad
- spotify
- lock
